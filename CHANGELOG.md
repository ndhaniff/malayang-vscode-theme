# Change Log

All notable changes to the "melayang-dark-theme" extension will be documented in this file.

Check [Keep a Changelog](http://keepachangelog.com/) for recommendations on how to structure this file.

## [0.6.0]

- added `Malayang Eclipse Theme`, a new variant darker than Midnight
- tuned the new darkest variant for ultra-dark environments while keeping syntax readable

## [0.5.0]

- added a matching `Malayang Light Theme`
- refined all variants for better readability and lower eye strain
- rebuilt token colors around a more consistent cross-language color system
- improved semantic highlighting and workbench color coverage

## [0.4.0]

- completed the main theme with broader VS Code workbench color coverage
- added richer semantic token colors
- added a new much darker variant: `Malayang Midnight Theme`

## [0.3.0]

- Initial release
